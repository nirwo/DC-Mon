CSV Structure

Your CSV file should have the following columns:

  • team (required): Team name
  • name (required): Application name
  • host (required): Host address
  • port (optional): Port number
  • webui_url (optional): Web UI URL
  • shutdown order (optional): Order for shutdown (number)
  • dependency (optional): Comma-separated list of dependent applications
Example CSV: 
team,name,host,port,webui_url,shutdown order,dependency
DevOps,MongoDB,mongodb-server,27017,http://mongodb-server:8080,1,
DevOps,Web Server,web-server,80,http://web-server,2,MongoDB
DevOps,Load Balancer,lb-server,443,http://lb-server,3,Web Server
