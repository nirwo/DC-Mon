Import Applications
Step 1: Select CSV File
Your CSV should include these columns: name, team, host, port (optional), webui_url (optional), db_host (optional), shutdown_order (optional), dependencies (optional)
CSV Format Guide

Required Fields:
- name: Application name
- team: Team name
- host: Server hostname

Optional Fields:
- port: Port number
- webui_url: Web UI URL
- db_host: Database host
- shutdown_order: Order (1-100)
- dependencies: Semicolon-separated list

Step 2: Import Options
Replace All mode will delete all existing applications and their dependencies.
